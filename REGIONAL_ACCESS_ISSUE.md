# Проблема: Сайт работает только на вашем WiFi/интернете

## Возможные причины:

### 1. Кеш Vercel CDN (наиболее вероятно)
Vercel кеширует старую версию сайта для других регионов/локаций.

**Решение:**
1. Vercel Dashboard → Deployments
2. Найдите последний деплой
3. Нажмите **три точки (⋮)** → **Redeploy**
4. **ВАЖНО:** Если есть опция "Clear Build Cache" или "Use existing Build Cache" - снимите галочку
5. Выполните Redeploy
6. Подождите 5-10 минут, чтобы CDN обновился глобально

### 2. Переменные окружения не применяются глобально
Переменные окружения на Vercel могут кешироваться по регионам.

**Решение:**
1. Убедитесь, что переменные установлены для **Production** (не только Preview/Development)
2. После изменения переменных выполните **Redeploy**
3. Подождите несколько минут

### 3. Проблема с DNS
DNS может кешироваться по-разному в разных локациях.

**Решение:**
- На проблемных устройствах очистите DNS кеш
- ИЛИ используйте другой DNS (например, Google DNS: 8.8.8.8)

### 4. Региональные ограничения Supabase
Supabase может иметь региональные ограничения или проблемы с доступом из некоторых регионов.

**Проверка:**
1. Откройте Supabase Dashboard
2. Settings → API → проверьте, нет ли региональных ограничений
3. Проверьте, доступен ли Supabase URL из разных локаций (можно использовать онлайн-сервисы для проверки)

### 5. Проблема с SSL/сертификатом
Могут быть проблемы с SSL-сертификатом для некоторых регионов.

**Проверка:**
- Откройте сайт на проблемном устройстве
- Проверьте, нет ли ошибок SSL в консоли браузера

## Рекомендуемый порядок действий:

1. **Выполните Redeploy на Vercel без кеша**
2. **Подождите 10-15 минут** (CDN обновляется глобально не мгновенно)
3. **Проверьте на проблемном устройстве в режиме инкогнито**
4. **Проверьте логи Vercel** - нет ли ошибок

## Диагностика:

На проблемном устройстве:
1. Откройте сайт в режиме инкогнито
2. Откройте DevTools (F12) → Network
3. Обновите страницу
4. Проверьте, какие запросы не выполняются (красные)
5. Посмотрите, к каким URL идут запросы (должны быть к вашему Supabase URL)

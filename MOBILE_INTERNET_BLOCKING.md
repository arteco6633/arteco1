# Проблема: Сайт не работает на мобильном интернете

## Симптомы:
- ✅ На ПК с WiFi - работает
- ✅ На телефоне с WiFi - работает  
- ❌ На телефоне с мобильным интернетом (без WiFi) - НЕ работает

## Причина: Мобильный оператор блокирует запросы

Мобильный оператор может блокировать или ограничивать запросы к Supabase по разным причинам.

## Решения:

### 1. Проверка на мобильном интернете
На телефоне с мобильным интернетом (БЕЗ WiFi):
1. Откройте сайт
2. Откройте DevTools (если возможно) или используйте удаленную отладку
3. Перейдите на вкладку Network
4. Обновите страницу
5. Найдите запросы к Supabase (`iwamliovlphpfnghqngn.supabase.co`)
6. Посмотрите статус ответов:
   - Если 503/504 - оператор блокирует
   - Если "Failed to fetch" - оператор блокирует
   - Если timeout - оператор блокирует или ограничивает

### 2. Проверка через VPN
На телефоне с мобильным интернетом:
1. Установите VPN (любой бесплатный)
2. Подключите VPN
3. Откройте сайт
4. Если работает с VPN - значит оператор блокирует

### 3. Проверка через другой мобильный оператор
Если есть возможность:
- Попробуйте с SIM-картой другого оператора
- Если работает - ваш оператор блокирует

### 4. Обращение к мобильному оператору
Если оператор блокирует:
- Обратитесь в техподдержку оператора
- Скажите, что сайт не работает на мобильном интернете
- Попросите проверить блокировку домена `iwamliovlphpfnghqngn.supabase.co`

## Временное решение:
Использовать VPN на мобильных устройствах (но это не решение для всех пользователей)

## Постоянное решение:
1. Проверить, не заблокирован ли домен Supabase вашим оператором
2. Если заблокирован - обратиться к оператору
3. ИЛИ использовать прокси/CDN для обхода блокировок (но это сложно)

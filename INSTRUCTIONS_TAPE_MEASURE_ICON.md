# Инструкция: Добавление иконки рулетки

## Текущее состояние
Иконка рулетки заменена на компонент `Image`, который загружает файл из папки `public/tape-measure-icon.svg`.

## Что нужно сделать:

1. **Скачайте SVG иконку рулетки:**
   - Зайдите на один из ресурсов:
     - [Flaticon - иконки рулетки](https://www.flaticon.com/ru/free-icon/measuring-tape_2769704)
     - [Icons8 - иконки рулетки](https://icons8.ru/icons/set/рулетка)
     - [Freepik - иконки рулетки](https://ru.freepik.com/icons/рулетка)
   
2. **Выберите подходящую иконку:**
   - Предпочтительно: изометрическая рулетка (как на референсе)
   - Формат: SVG
   - Стиль: минималистичная, белые линии на прозрачном фоне

3. **Сохраните файл:**
   - Название файла: `tape-measure-icon.svg`
   - Расположение: `/public/tape-measure-icon.svg`

4. **Проверка:**
   - Файл должен быть доступен по адресу: `http://localhost:3000/tape-measure-icon.svg`
   - Иконка автоматически инвертируется в белый цвет благодаря CSS фильтру

## Альтернатива: Встроенный SVG

Если хотите встроить SVG прямо в код вместо файла, замените компонент `Image` на встроенный `<svg>`. 

Файл находится в: `app/product/[id]/page.tsx`, строки ~1708-1717

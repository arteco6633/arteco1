# Добавление переменных окружения T-Bank в Vercel

## Шаги:

1. Перейдите в ваш проект на Vercel: https://vercel.com/dashboard

2. Откройте настройки проекта:
   - Нажмите на название проекта
   - Перейдите в раздел **Settings**
   - Выберите **Environment Variables** в меню слева

3. Добавьте следующие переменные:

   **Для Production, Preview и Development:**

   ```
   TBANK_TERMINAL_ID=1763021044538DEMO
   TBANK_PASSWORD=Nfj!60bSZi1VCv*n
   TBANK_API_URL=https://securepay.tinkoff.ru/v2/Init
   NEXT_PUBLIC_SITE_URL=https://www.arteeeco.ru
   ```

   ⚠️ **Важно:** 
   - Для `NEXT_PUBLIC_SITE_URL` убедитесь, что указан правильный домен вашего сайта на Vercel
   - Убедитесь, что в конце URL нет слеша `/` (используйте `https://www.arteeeco.ru` вместо `https://www.arteeeco.ru/`)

4. После добавления всех переменных:
   - Нажмите **Save**
   - Перезапустите деплой (Redeploy) или дождитесь следующего автоматического деплоя

5. После деплоя проверьте работу оплаты на сайте

## Для продакшена:

Когда получите реальные данные от T-Bank, замените тестовые значения:
- `TBANK_TERMINAL_ID` - ваш реальный Terminal ID
- `TBANK_PASSWORD` - ваш реальный Secret Key

## Проверка:

После деплоя проверьте:
1. Создайте тестовый заказ
2. Выберите "Оплата картой онлайн"
3. Должен произойти редирект на страницу оплаты T-Bank
